🍲 Comida de Mãe - Cardápio Digital & Delivery

Uma aplicação web completa de cardápio digital desenvolvida para facilitar pedidos e a gestão de um restaurante. O projeto permite que os clientes escolham pratos, gerenciem um carrinho de compras e enviem o pedido finalizado diretamente para o WhatsApp do estabelecimento.

🚀 Funcionalidades Principais

- **Catálogo Dinâmico:** Separação inteligente entre pratos principais e bebidas.
- **Carrinho de Compras:** Sistema persistente (LocalStorage) que permite adicionar, remover e alterar quantidades.
- **Modal de Detalhes:** Visualização ampliada do produto antes da compra.
- **Checkout Integrado:** Coleta de dados de entrega (Nome, Endereço, Telefone).
- **Integração WhatsApp:** Geração automática de mensagem formatada com os itens do pedido e dados do cliente.
- **Design Responsivo:** Interface otimizada para dispositivos móveis e desktop usando Tailwind CSS.

🛠 Tecnologias Utilizadas

Front-end & Estilização
* **Tailwind CSS:** Utilizado para toda a estilização e layout responsivo, garantindo agilidade e consistência visual.
* **HTML5 & CSS3:** Estrutura semântica e customizações específicas (como o efeito de scroll no footer).
* **JavaScript (Vanilla):** Lógica de manipulação do DOM, gerenciamento de estado do carrinho e integração com a API do WhatsApp.

Persistência & Ferramentas
* **LocalStorage:** Para manter os itens no carrinho mesmo se o usuário fechar o navegador.
* **Google Fonts:** Tipografia "Roboto" para uma leitura clara e profissional.


🎨 Branding & Identidade Visual
Diferente de muitos projetos de programação, o Comida de Mãe foi concebido como uma marca completa. Como Designer, desenvolvi:

Criação de Logotipo: Desenvolvida para transmitir acolhimento e tradição familiar.

Paleta de Cores Estratégica: 

#7D0A0A (Vinho/Vermelho): Estimula o apetite e remete à paixão pela culinária.
#03A791 (Verde): Traz o frescor dos ingredientes naturais.
#EFB036 (Amarelo/Dourado): Cor de destaque para botões de ação (CTA).

Pattern Design: Implementação de um padrão visual (logo-pattern) no rodapé para reforçar a presença da marca durante a navegação.
Iconografia: Seleção de ícones que facilitam a leitura rápida do cardápio e do carrinho.
